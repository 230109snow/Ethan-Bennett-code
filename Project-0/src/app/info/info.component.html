Bored API

This api will give you an activity to do based on the criteria you select.

You can find to do based on the type, price, accessibility, or number of participants

How To Use

First select what to base the search off of

If you choose to search by the type simply select the type you want to search by

If you choose accessibility or price enter a number between 0 and 100

The higher the number the more pricey or less accessible it is

This number must be a multiple of 5 though
